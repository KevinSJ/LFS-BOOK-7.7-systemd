6.46. Diffutils-3.3

Diffutils 软件包包含显示文件和目录差异的程序。

大概编译时间： 0.4 SBU
需要磁盘空间： 30 MB

6.46.1. 安装 Diffutils

首先修复文件安装区域文件： 

sed -i 's:= @mkdir_p@:= /bin/mkdir -p:' po/Makefile.in.in

准备编译 Diffutils： 

./configure --prefix=/usr

编译软件包： 

make

用以下命令测试结果： 

make check

安装软件包： 

make install

6.46.2. Diffutils 软件包内容

安装的程序： cmp, diff, diff3, 和 sdiff

简要介绍

cmp

比较两个文件并报告字节差异

diff

比较两个文件或目录并报告文件中的行差异

diff3

逐行比较三个文件

sdiff

比较两个文件并交互式输出结果
