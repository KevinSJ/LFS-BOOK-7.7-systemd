6.51. Gperf-3.0.4

Gperf 为键集合生成完美的哈希函数。

大概编译时间： 少于 0.1 SBU
需要磁盘空间： 6.0 MB

6.51.1. 安装 Gperf

准备编译 Gperf： 

./configure --prefix=/usr --docdir=/usr/share/doc/gperf-3.0.4

编译软件包： 

make

用以下命令测试结果： 

make check

安装软件包： 

make install

6.51.2. Gperf 软件包内容

安装的程序： gperf
