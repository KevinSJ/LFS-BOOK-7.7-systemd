5.24. Grep-2.21

Grep 软件包包含了在文件中搜索的工具。

大概编译时间： 0.2 SBU
所需磁盘空间： 20 MB

5.24.1. 安装 Grep

配置 Grep 准备编译： 

./configure --prefix=/tools

编译软件包： 

make

现在编译已经完成。之前说过，不要求为本章中所构建的临时工具运行测试套件。不过你仍然要测试 Grep 的话可以用下面的命令： 

make check

安装软件包： 

make install

关于这个软件包的详细资料请查看本书章节：6.34.2, “Grep 软件包内容”
